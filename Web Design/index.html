Manipulating Form Data

Here we have a simplified HTML form, which will take user input and upon submission, do something with it using JavaScript. This example makes use of pattern recognition, required fields, and multiple types of input items to showcase how a common form might work on your webpage.

This form is a smart form: it will not let you submit any data that doesn't match its predefined patterns and requirements. In this case, you must enter a value for first and last name, the middle initial may only be a single character, and only characters A-Z upper- or lower-case are accepted as input.

Once these conditions are met, press the 'Submit' button and a personalized message will be displayed in the box below.
